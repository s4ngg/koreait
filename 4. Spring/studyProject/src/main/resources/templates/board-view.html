CSS Flexbox와 Grid의 차이점 정리

김개발 2026-02-10 조회 42

CSS에서 레이아웃을 구성할 때 가장 많이 사용되는 두 가지 방법은 Flexbox와 Grid입니다. 각각의 특징과 사용 시나리오를 정리해 보겠습니다.

1. Flexbox (1차원 레이아웃)

Flexbox는 한 방향(가로 또는 세로)으로 요소를 배치할 때 유용합니다. 네비게이션 바, 카드 리스트, 정렬 등 단일 축 레이아웃에 적합합니다.

주요 속성: display: flex, justify-content, align-items, flex-wrap

2. Grid (2차원 레이아웃)

Grid는 행과 열을 동시에 제어할 수 있는 2차원 레이아웃 시스템입니다. 복잡한 페이지 레이아웃, 대시보드, 갤러리 등에 적합합니다.

주요 속성: display: grid, grid-template-columns, grid-template-rows, gap

3. 결론

단순한 정렬이나 1차원 배치에는 Flexbox를, 복잡한 2차원 레이아웃에는 Grid를 사용하는 것이 좋습니다. 두 가지를 함께 사용하면 더욱 유연한 레이아웃을 구성할 수 있습니다.
